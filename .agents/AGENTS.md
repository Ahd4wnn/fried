# Workspace Rules & Preferences

## Profile Card Design Guidelines ("Dream Design")

When designing therapist profiles, seeker profiles, or matched cards, always implement the following design pattern:

### 1. Aesthetic Styles
- **Vibrant Gradient Banner / Background**: Use a premium green gradient mesh (incorporating Hovio's forest green `#1C5C32` and lighter/darker shades) with a subtle noise/grain pattern.
- **Glassmorphism**: When using full gradient backgrounds, pair them with white text, translucent outline buttons, and high-contrast elements.
- **Contrast**: For light cards, use a white background with the gradient restricted to a top header banner.

### 2. Layout Structure
- **Circular Overlapping Avatar**: Render a clean circular profile photo or initials placeholder. On banner layouts, let the avatar overlap the bottom border of the banner.
- **Metadata Alignment**: Align the name (bold/medium sans-serif, e.g. Inter) and title/handle underneath it. Action buttons (like Book, Shortlist) should sit on the right of the profile header.
- **Stats Row**: Include a stats section showing metrics (e.g., Rating, Price, Years of Experience) using clean icons and stacked labels (metric value above, desaturated name label below).
- **Arrow CTA Button**: At the very bottom of the card, place a full-width pill-shaped button containing a circular arrow icon on the left (`←` or `→` inside a circle) and a centered action text (e.g. "Get in Touch" or "Book Session").

---

## Typography Guardrails
- **Instrument Serif (`font-display`)**: Do **NOT** add weight (such as `font-bold` or `font-semibold`) to any headings utilizing the Instrument Serif font. Keep them at `font-normal` or `font-light` to preserve their elegance.
